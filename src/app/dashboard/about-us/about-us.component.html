OpenWeather - это команда ИТ-экспертов и специалистов по обработке данных, которая занимается глубоким изучением данных о погоде с 2014 года. Для каждой точки земного шара OpenWeather предоставляет исторические, текущие и прогнозируемые данные о погоде через API-интерфейсы со скоростью света. Штаб-квартира в Астане, Казахстан.

Агро API

Наша новая технология Time Machine позволила нам улучшить данные в Исторической коллекции погоды : исторические данные о погоде теперь доступны для любых координат, а глубина исторических данных увеличена до 40 лет.

Как получить
Торговая площадка готовых наборов данных (города, почтовые индексы, сетки)

Наборы
на лету для настраиваемых списков координат

API (для города, до 1 года назад; подписки с различными ограничениями на звонки в минуту, доступность данных и обслуживание)
